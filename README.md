# C++ Winsock Client Architecture

# Client Functions TCP 

initialize WSA - WSAStartup()
Create a socket - socket()
Connect to the server - connect()
Send and recive data - recv(), send(), recvfrom(), sendto()
Disconnect - closesocket()
